# 🌿 Tarang Hazaribagh — NGO Website

A fully functional, responsive NGO donation website built for **Tarang Hazaribagh** — a non-profit organization dedicated to uplifting underprivileged communities across Jharkhand through education, healthcare, and sustainable livelihoods.

---

## 🔗 Live Preview

> Open `index.html` in any browser to view the website locally.

---

## 📋 Project Overview

This project was built as part of my **Web Development Internship**. The goal was to create a professional, user-friendly website for Tarang Hazaribagh NGO that allows visitors to learn about the organization and donate money easily.

## 💳 Donation Flow

1. User visits the Donate section
2. Selects preset amount or enters custom amount
3. Fills in personal details (Name, Email, Phone, PAN)
4. Clicks "Donate Now" button
5. Form validates all fields
6. Success popup confirms donation with 80G receipt info

**OR**

1. User scans QR code with any UPI app
2. Enters amount
3. Completes payment instantly
4. Sends screenshot to WhatsApp number for receipt
